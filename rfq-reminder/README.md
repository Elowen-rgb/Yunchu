# 询价提醒助手

Chrome/Edge 浏览器插件，专为供应商老板设计。自动识别采购/询价/招投标网页中的项目信息，截止前 48 小时高频微信提醒，再也不会错过报价截止时间。

## 功能概览

- 🔍 **一键识别** — 打开任意采购网页，点击按钮自动提取询价标题、发布人、单号、截止时间
- ⏰ **智能提醒** — 仅截止前 48 小时内提醒，越临近频率越高（最高每 10 分钟一次）
- 📱 **微信推送** — 支持 WxPusher / Server酱 / 自定义 Webhook，提醒直达个人微信
- 📊 **项目管理** — popup 面板查看所有项目，标记已报价/放弃，按紧急程度排序
- 🔄 **自动去重** — URL 相同或标题+截止时间相同的项目不会重复保存
- ⚙️ **灵活配置** — 自定义提醒间隔、关键词、推送服务

## 快速开始

### 1. 加载插件

1. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions/`
2. 右上角打开「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `rfq-reminder` 文件夹
5. 插件图标出现在浏览器右上角

> Edge 浏览器：`edge://extensions/`，操作相同。

### 2. 配置微信推送（可选）

插件默认使用浏览器通知。如需微信提醒：

**WxPusher（推荐）：**

1. 打开 https://wxpusher.zjiecode.com
2. 注册 → 创建应用 → 获取 AppToken
3. 关注 WxPusher 公众号 → 获取 UID
4. 在插件 Options 页面填入 AppToken 和 UID
5. 点击「发送测试消息」验证

**Server酱：**

1. 打开 https://sct.ftqq.com
2. 注册 → 获取 SendKey
3. 在 Options 页面填入 SendKey

### 3. 开始使用

1. 打开任何采购/询价/招投标网页
2. 点击浏览器右上角插件图标
3. 点击「识别本页询价」
4. 项目自动保存到本地，到时间自动提醒

## 如何测试

### 测试页面

用浏览器打开 `test-pages/inquiry-sample.html`，该页面包含 6 个不同格式的模拟询价项目。

```bash

node tests/parser.test.js     # 页面字段识别测试（10项）
```

### 手动测试步骤

1. 打开 `test-pages/inquiry-sample.html`
2. 点击插件图标 → 点击「识别本页询价」
3. 确认 popup 中显示项目信息
4. 尝试「已报价」「放弃」「删除」按钮
5. 打开 Options 页面，配置微信推送并发送测试消息

## 提醒逻辑

| 剩余时间 | 提醒间隔 |
|----------|----------|
| 48 小时内 | 每 24 小时 |
| 24 小时内 | 每 12 小时 |
| 12 小时内 | 每 6 小时 |
| 6 小时内 | 每 2 小时 |
| 2 小时内 | 每 30 分钟 |
| 30 分钟内 | 每 10 分钟 |
| 已过期未处理 | 提醒一次 |

- 已标记「已报价」或「放弃」的项目不再提醒
- 距离截止超过 48 小时的项目不提醒
- 间隔可在 Options 页面自定义

## 为什么使用第三方推送服务

本插件使用 WxPusher / Server酱 等第三方公众号消息推送服务，原因如下：

1. **合规性** — 不涉及个人微信号登录、不控制微信客户端、不抓取微信数据
2. **稳定性** — 不依赖网页版微信（已停止服务）、不依赖微信机器人框架
3. **安全性** — 不会触发微信风控导致封号
4. **便捷性** — 只需关注公众号即可接收推送，无需额外客户端

⚠️ **本插件不会：**
- 自动登录个人微信
- 控制微信客户端
- 抓取微信聊天记录
- 使用任何网页版微信 API

## 权限说明

| 权限 | 用途 |
|------|------|
| `storage` | 本地保存项目数据和配置 |
| `notifications` | 浏览器弹窗提醒 |
| `alarms` | 定时检查项目到期状态 |
| `activeTab` | 读取当前标签页内容 |
| `scripting` | 注入脚本识别页面询价信息 |
| `tabs` | 获取标签页标题和 URL |
| `host_permissions: <all_urls>` | 需要在用户打开的任意采购网页中读取内容。后续版本将改为用户手动配置域名白名单。 |

## 当前限制

- **MVP 版本**：同时识别一个页面的主要询价信息，暂不支持批量识别表格中多条项目
- **网站适配**：依赖页面中的关键词匹配，部分非标准格式可能识别失败
- **微信推送**：需用户自行注册 WxPusher / Server酱 账号
- **浏览器通知**：需浏览器保持运行状态
