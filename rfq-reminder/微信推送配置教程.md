# 🐱 小猫询价插件 —— 微信推送配置教程

**配置后效果：** 询价项目截止前，微信自动收到提醒，越临近截止提醒越频繁。

---

## 第一步：注册 WxPusher

1. 浏览器打开 👉 **[https://wxpusher.zjiecode.com](https://wxpusher.zjiecode.com)**
2. 用微信扫描网页上的二维码
3. 登录成功后进入管理后台

## 第二步：创建应用获取 AppToken

1. 左侧菜单点击 **「应用管理」**
2. 点击 **「新增应用」**
3. 随便填写：
   - 应用名称：`询价提醒`
   - 其他随便填或留空
4. 点击确定，页面会显示一个 **AppToken**
5. 📋 **复制 AppToken 保存好**

![示意：应用管理页面](https://wxpusher.zjiecode.com/docs/)

## 第三步：获取你的 UID

1. 左侧菜单点击 **「用户管理」**
2. 你会看到自己的微信信息
3. 旁边有一串类似 `UID_xxxxxxxxxxxxxxxx` 的字符 —— 这就是你的 **UID**
4. 📋 **复制 UID 保存好**

> ⚠️ 如果「用户管理」里是空的：点「关注推送」→ 用微信扫码关注 WxPusher 公众号 → 刷新页面 → UID 就出现了

## 第四步：让同事也收到提醒（可选）

1. 让同事用微信关注 **WxPusher 公众号**
2. 同事把他们的 UID 发给你
3. 所有 UID 用英文逗号拼在一起：
   ```
   UID_xxxxx, UID_yyyyy, UID_zzzzz
   ```

## 第五步：填入插件

1. 点击浏览器右上角小猫图标 🐱
2. 点击底部 **「更多设置」** 打开设置页
3. 选择推送服务：**WxPusher（推荐）**
4. 粘贴 **AppToken** 和 **UID**
5. 点击 **「💾 保存所有设置」**
6. 点击 **「📤 发送测试消息」**
7. 打开微信，应该能看到测试消息 🎉

---

## 常见问题

| 问题 | 解决方法 |
|---|---|
| **收不到测试消息** | 确认已关注 WxPusher 公众号，UID 格式正确（UID_ 开头） |
| **AppToken 忘记了** | 登录 WxPusher → 应用管理 → 查看 |
| **换微信号了** | 重新关注公众号，新的 UID 会出现在用户管理中 |
| **同事收不到** | 确认同事已关注公众号，且 UID 正确填入 |

---

## 注意

- 本插件通过 WxPusher 公众号推送，**不涉及个人微信号操作**，安全合规
- WxPusher **完全免费**，无需付费
- 每个 WxPusher 应用每天有 **1000 条**消息额度，完全够用
